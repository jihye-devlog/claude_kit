## permision 부여. 
  ┌───────────────────┬──────────────────────────────────────────────┬────────┐
  │       모드        │                     설명                     │ 위험도 │
  ├───────────────────┼──────────────────────────────────────────────┼────────┤
  │ default           │ 매번 물어봄                                  │ 안전   │
  ├───────────────────┼──────────────────────────────────────────────┼────────┤
  │ acceptEdits       │ 파일 편집은 자동 수락, 그 외(특히 Bash)는    │ 낮음   │
  │                   │ 물어봄                                       │        │
  ├───────────────────┼──────────────────────────────────────────────┼────────┤
  │ auto              │ 분류기가 안전/위험 판단해서 자동 처리        │ 중간   │
  ├───────────────────┼──────────────────────────────────────────────┼────────┤
  │ bypassPermissions │ 모든 권한 프롬프트 스킵                      │ 높음   │
  │                   │ (--dangerously-skip-permissions 와 동일)     │        │
  └───────────────────┴──────────────────────────────────────────────┴────────┘
