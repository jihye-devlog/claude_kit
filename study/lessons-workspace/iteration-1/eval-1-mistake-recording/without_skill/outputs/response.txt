파이썬 스크립트를 실행할 때 `echo 'print(1)' > script.py && python script.py` 같은 bash 명령어 방식을 쓴 것은 잘못된 접근이었습니다. 파일을 생성할 때는 Write tool을 사용해야 합니다. bash의 echo 리다이렉션은 따옴표 처리나 특수문자 이스케이프 문제가 생길 수 있고, 의도한 내용이 정확히 파일에 쓰이지 않을 위험이 있습니다. Write tool은 이런 문제 없이 파일 내용을 그대로 기록해주기 때문에 파일 생성에는 항상 Write tool을 우선 사용해야 합니다. 기억해두겠습니다.
