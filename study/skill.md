## loop
### 특정 동작을 특정 주기동안 반복.
- 예를 들어 에이전트 팀 사용할때 tmux 에러또는 각 팀원들이 idle 상태인데도 불구하고 작업을 계속 진행을 하지 않는 경우가 생긴다.
- 이때 하기 커맨드로 사용하면 좋음!
```
/loop 3m 팀원 상태를 점검해줘. 점검 방법: 1) TaskList로 현재 상태 확인 2) in_progress 또는 pending(unblocked) 태스크가 있는 팀원에게는 무조건 SendMessage로 "alive? 현재 작업 중인 태스크와 진행상황을 알려줘. 멈춰있으면 즉시 재개해." 를 보내 3) 작업이 전부 completed인 팀원은 무시 4) 태스크 상태가 in_progress인데 이전 점검과 동일한 태스크에 머물러 있으면 "멈춘 것 같다. 지금 바로 계속 진행해줘. 막힌 부분이 있으면 알려줘." 로 깨워줘 5) 모든 태스크가 completed면 loop 종료
```
