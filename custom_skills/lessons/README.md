# lessons

Claude가 저지른 실수를 `~/.claude/mistakes.md`에 기록하고 반복을 방지하는 스킬입니다.

## 설치 방법

스킬 파일을 글로벌 커맨드 디렉토리에 복사합니다.

```bash
cp lessons/lessons.md ~/.claude/commands/lessons.md
```

Claude Code를 재시작하면 `/lessons` 슬래시 커맨드로 사용할 수 있으며, 실수 상황에서 자동으로도 동작합니다.

## 사용 방법

### 자동 기록

아래 상황에서 Claude가 자동으로 `~/.claude/mistakes.md`에 기록합니다.

- 사용자가 교정 발언을 할 때 ("아니야", "틀렸어", "그게 아니라" 등)
- Claude가 작성한 코드나 명령이 에러를 유발했을 때
- Claude가 작업 도중 이전 단계의 실수를 스스로 발견했을 때

### 슬래시 커맨드

```
/lessons
```

지금까지 기록된 실수 목록을 날짜순으로 요약해서 보여줍니다.

## 동작 방식

| 단계 | 내용 |
|------|------|
| 감지 | 사용자 교정, 빌드 실패, 자기 인식 등 실수 상황 판단 |
| 기록 | ~/.claude/mistakes.md 에 날짜, 실수 내용, 교정 방법, 관련 파일 추가 |
| 세션 시작 | mistakes.md 가 있으면 읽어 과거 실수 파악 후 반복 방지 |
| /lessons | 전체 실수 목록 요약 출력 및 반복 패턴 강조 |

## 저장 위치

```
~/.claude/mistakes.md
```

전역 파일로 모든 프로젝트에서 공유됩니다.
